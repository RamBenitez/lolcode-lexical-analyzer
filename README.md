# LOLCODE Interpreter

A full-featured LOLCODE interpreter with lexical analysis, syntax parsing, and semantic execution.

## Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/RamBenitez/lolcode-lexical-analyzer.git
   cd lolcode-lexical-analyzer
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

## How to Run

### GUI Version
1. Navigate to the `gui` directory:
   ```bash
   cd gui
   ```

2. Run the GUI application:
   ```bash
   python app.py
   ```

### Command Line Version
1. Navigate to the `src` directory:
   ```bash
   cd src
   ```

2. Run the main program (uses `tests/testcase.lol`):
   ```bash
   python main.py
   ```
